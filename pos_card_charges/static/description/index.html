Odoo 11 POS Card Charges.

This module allow us to take payment charges on order.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • In the below screen shot we can see a payment charge input box.
  • In this box we can set charge amount on payment.


  • Now in receipt we can see a product for charges and charge under the journals.
  • Now here in payment methods configuration we can set configuration for payment charge.
